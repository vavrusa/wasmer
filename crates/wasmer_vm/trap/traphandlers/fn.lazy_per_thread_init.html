pub fn lazy_per_thread_init() -> Result<(), Trap>

A module for registering a custom alternate signal stack (sigaltstack).

Rust's libstd installs an alternate stack with size SIGSTKSZ, which is not always large enough for our signal handling code. Override it by creating and registering our own alternate stack that is large enough and has a guard page.
