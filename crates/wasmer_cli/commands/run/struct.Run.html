The options for the wasmer run subcommand

Fields

disable_cache: bool

Disable the cache

path: PathBuf

File to run

invoke: Option<String>

Invoke a specified function

command_name: Option<String>

The command name is a string that will override the first argument passed to the wasm program. This is used in wapm to provide nicer output in help commands and error messages of the running wasm program

cache_key: Option<String>

A prehashed string, used to speed up start times by avoiding hashing the wasm module. If the specified hash is not found, Wasmer will hash the module as if no cache-key argument was passed.
